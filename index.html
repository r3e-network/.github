Key Projects

Neo N3 C++ Full Node

Enterprise Performance Full Node

High-performance, production-ready Neo N3 full node implementation in C++ for maximum efficiency and system-level optimization. Features enterprise-grade performance, memory management, and native C++ APIs for blockchain integration.

C++ Full Node Enterprise Performance

Neo N3 Rust Full Node

Memory-Safe Modern Full Node

Modern, memory-safe Neo N3 full node implementation in Rust, designed for reliability, security, and developer experience. Features WebAssembly plugin system, built-in observability, and async runtime for high performance.

Rust Full Node Memory Safe Async

Neo N3 MCP Server

AI-Native Blockchain Integration

An MCP server providing 34 tools and 9 resources for comprehensive Neo N3 blockchain interaction. Features network management, wallet operations, smart contract interactions, and blockchain data querying with security-first design.

TypeScript MCP AI Integration Blockchain

Neo Rust Smart Contract DevPack

Rust for Smart Contracts

Enable developers to write Neo N3 smart contracts using Rust, bringing memory safety and performance to Neo blockchain development with familiar syntax and comprehensive development toolkit.

Rust Smart Contracts Neo N3 DevTools

Neo Rust SDK

Multi-Chain Rust Development

A powerful SDK for interacting with both Neo N3 and Neo X blockchains using Rust. Features multi-blockchain support, type-safe interactions, high-performance RPC client, and comprehensive wallet management.

Rust SDK Neo N3 Neo X

OpenR3E

Secure Contract Libraries for dApps

Comprehensive collection of secure, audited smart contract libraries for Neo N3 dApp development. Provides battle-tested components for tokens, DeFi, access control, and utilities with 50% gas optimization and security best practices.
